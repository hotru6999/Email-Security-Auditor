## 🛠️ How to Use

Launch the application using the icon on your desktop or start menu. The interface shows a main dashboard. Use the tabs to navigate between the auditor, the credential manager, and the report viewer.

### Managing Lists
Load your list of email addresses from a text file. Ensure each address and password pair rests on a new line. You can choose to format your list using commas or colons as separators. The application detects common formats automatically.

### Running an Audit
Select an email account or a full list from your imported data. Click the start button to begin the security check. The tool connects to your email server using secure protocols. It tests if the credentials grant access to the account. It also checks if the account has two-factor authentication enabled or if it relies on older login methods.

### Viewing Results
The application displays a live feed of the audit process. Green marks indicate successful access, while red marks show failed attempts or insecure settings. You can export these results as a text file for your records. Open the settings menu to save your reports to a folder on your drive.

## 📉 Security Features

This tool includes various features to help you understand email account safety.

*   **Credential Validation:** Confirms if login pairs remain functional.
*   **Authentication Check:** Detects the presence of multi-factor protections.
*   **Protocol Analysis:** Identifies if accounts use secure SMTP or IMAP connection settings.
*   **Security Auditing:** Measures the strength of your password policies.

## ❓ Troubleshooting

Common issues often relate to basic settings. Review these tips if you face trouble during the audit process.

*   **Connection Errors:** Ensure your internet connection is stable. Some email providers limit how many times a tool can check a password in a short time. Wait ten minutes before you try again.
*   **Blocked Access:** Some providers block third-party tools by default. You may need to go to your email account security page online and generate an app password. Use this app password in the software instead of your regular password.
*   **Firewall Warnings:** Your computer may show a alert when you first open the tool. Select the option to allow the software to communicate on private networks.
*   **Missing Files:** If the program fails to launch, download and install the .NET Desktop Runtime from the official Microsoft support page.

## 📜 Usage Policy

Use this tool only for accounts you own or have explicit permission to audit. The software helps you manage your security posture, but you retain responsibility for how you use your credentials. Keep your login data in an encrypted file to prevent unauthorized access. The developer does not store your credentials. All checks happen locally on your machine.
